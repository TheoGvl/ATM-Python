# ATM Pin Pad Οθόνη Κωδικού ΑΤΜ 

Μια μοντέρνα και διαδραστική προσομοίωση οθόνης ξεκλειδώματος (PIN Pad) τύπου ΑΤM. Το project αυτό δημιουργήθηκε για να εξασκηθώ στον προσαρμοσμένο σχεδιασμό UI (Custom Components) και στη διαχείριση κατάστασης (State Management) με τη βιβλιοθήκη Flet στην Python.

## Τι κάνει;
* **Διαδραστικό Πληκτρολόγιο:** Διαθέτει ολοστρόγγυλα κουμπιά 0-9 με εφέ "splash" ink όταν τα πατάς.
* **Κρυπτογράφηση Οθόνης:** Καθώς ο χρήστης πληκτρολογεί τον κωδικό του, οι αριθμοί κρύβονται και μετατρέπονται δυναμικά σε αστεράκια (`✱`).
* **Έλεγχος Πρόσβασης:** Ο χρήστης μπορεί να πατήσει το πράσινο "OK" για να ελέγξει αν το PIN είναι σωστό (προεπιλογή: `1234`) ή το κόκκινο "C" για να καθαρίσει την οθόνη. Τα μηνύματα επιτυχίας και αποτυχίας αλλάζουν χρώμα ανάλογα με το αποτέλεσμα!


**Βασικές προγραμματιστικές τεχνικές που εφαρμόστηκαν:**
1. **Custom Components (DRY Principle):** Αντί να γράψω τον κώδικα για το κάθε κουμπί ξεχωριστά, έφτιαξα μια συνάρτηση (`ftiakse_koumpi`) η οποία "γεννάει" Containers με συγκεκριμένα χαρακτηριστικά (στρογγυλεμένες γωνίες, χρώματα, κλπ).
2. **Συντεταγμένες Alignment:** Χρήση του `ft.Alignment(0, 0)` για απόλυτο κεντράρισμα στοιχείων μέσα σε Containers, προσαρμοσμένο στις τελευταίες εκδόσεις της Flet.
3. **Data Attributes (`e.control.data`):** Αποθήκευση κρυφών δεδομένων πίσω από κάθε γραφικό στοιχείο για να αναγνωρίζει το πρόγραμμα ποιο ακριβώς πλήκτρο πατήθηκε.
4. **Nonlocal Variables:** Διαχείριση της μνήμης του κωδικού (state) μέσα από εμφωλευμένες συναρτήσεις.
